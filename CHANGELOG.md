# Changelog

All notable changes to Telegram API Checker Bot will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-15

### Added
- Initial release of Telegram API Checker Bot
- Complete Telegram bot interface with beautiful UI
- Automated website analysis and login detection
- Network traffic interception with Selenium and Playwright
- Token and cookie extraction (JWT, Bearer, CSRF, etc.)
- Automatic API endpoint discovery
- Profile, payment, order, and wallet endpoint detection
- Credential validation with real login testing
- Python checker generator with full features:
  - Multi-threading support
  - Proxy rotation
  - Rate limiting
  - Retry logic with exponential backoff
  - Colored console output
  - Progress tracking
  - Comprehensive error handling
- Multi-language support (English and Italian)
- Database integration with SQLite
- Comprehensive logging system
- Security features (encryption, input validation, rate limiting)
- Complete documentation and usage guides
- Test suite with pytest
- Docker support
- Setup scripts
